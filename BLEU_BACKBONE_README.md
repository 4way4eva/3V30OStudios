## Utility Commands

The `bleu_backbone_utils.js` script provides the following commands:

```bash
# Show top N products by ROI
node scripts/bleu_backbone_utils.js top-roi [n]

# Show top N products by overscale
node scripts/bleu_backbone_utils.js top-overscale [n]

# Show top N products by combined score (ROI √ó Overscale)
node scripts/bleu_backbone_utils.js top-combined [n]

# Show all products in a sector
node scripts/bleu_backbone_utils.js sector <sector_id>
# Example: node scripts/bleu_backbone_utils.js sector HLNG-MED-BIO

# Show all products in a domain
node scripts/bleu_backbone_utils.js domain <CIVILIAN|MILITARY|COSMIC>

# Search products by name, use case, or signal
node scripts/bleu_backbone_utils.js search <query>

# Show sector summary statistics
node scripts/bleu_backbone_utils.js summary

# Export all data to CSV
node scripts/bleu_backbone_utils.js export-csv

# Export report to Markdown
node scripts/bleu_backbone_utils.js export-md
```
