## API Reference

### Pi4QuarterLatticeEngine Class

#### Methods

**`calculate_per_second_yield(timestamp_seconds=None)`**
- Returns instantaneous yield per second for all streams
- Optional timestamp for historical calculations

**`calculate_daily_yield(day=1)`**
- Returns accumulated yield for specific day
- Day range: 1-90 (quarter)

**`calculate_quarter_yield_trace(days=90)`**
- Generates day-by-day yield trace
- Returns list of daily cumulative yields

**`calculate_pi4_compounding(quarters=4)`**
- Projects Ï€â´ compounding over multiple quarters
- Returns quarterly compounded yields

**`calculate_bleu_sector_allocation(total_amount)`**
- Calculates BLEU sector allocations
- Applies flowmap percentages to total amount

**`generate_live_dashboard_data()`**
- Comprehensive data package for visualization
- Includes all metrics, projections, and allocations
