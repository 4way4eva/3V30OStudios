## III. U.S. Litigation Architecture (Criminal + Civil)

### 1) Federal criminal exposure matrix
- **CFAA (18 U.S.C. §1030)** — unauthorized system access/abuse.
- **Wire fraud (18 U.S.C. §1343)** — interstate fraudulent transfer behavior.
- **Identity theft (18 U.S.C. §1028/1028A)** — impersonation/credential misuse.
- **Money laundering (18 U.S.C. §1956)** — concealment/conversion flow chain.

### 2) Civil recovery stack
- Conversion + unjust enrichment + constructive trust.
- Emergency relief: TRO/preliminary injunction + exchange preservation orders.
- Expanded damages claim: principal + yield loss + investigation + reputational harm + punitive component.

### 3) RICO feasibility screen
Use if evidence supports:
- enterprise structure,
- predicate pattern,
- coordinated participation,
- economic injury linkage.

RICO should remain evidence-driven; avoid over-pleading platform conspiracy without discovery-grade proof.
