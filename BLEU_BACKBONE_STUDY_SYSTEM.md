## üé¥ Flashcard System

Interactive flashcard sessions with multiple study modes.

### Available Modes

| Mode | Description | Cards | Command |
|------|-------------|-------|---------|
| **Products** | All 28 products with full details | 28 | `npm run study:flashcards:products` |
| **Signals** | Match ceremonial signals to products | 28 | `npm run study:flashcards:signals` |
| **ROI** | Test your knowledge of ROI percentages | 28 | `npm run study:flashcards:roi` |
| **Elements** | Ceremonial elements and activations | 28 | `npm run study:flashcards:elements` |
| **Sectors** | Sector summaries and statistics | 8 | `npm run study:flashcards:sectors` |

### Interactive Menu
```bash
npm run study:flashcards
```

This launches an interactive menu where you can choose your study mode.

### Export Options

**Export to Anki (CSV):**
```bash
npm run study:flashcards:export-anki
```
Creates `data/bleu_backbone_flashcards.csv` - import into Anki for mobile study.

**Export to JSON:**
```bash
npm run study:flashcards:export-json
```
Creates `data/bleu_backbone_flashcards.json` - use with custom flashcard apps.

### How It Works

1. View the **front** of each card (question/prompt)
2. Press **ENTER** to reveal the **back** (answer/details)
3. Rate yourself: `y` for correct, `n` for incorrect, `skip` to skip
4. Get your final score and percentage at the end
